# Portal para Desarrolladores de Terpa

Portal público con guías de integración, onboarding técnico y ejemplos del ecosistema Terpa.

## Capa
SDK Público de Sandbox y Documentación.

## Proposito
- Publicar documentación de integración para partners.
- Documentar el flujo sandbox y el uso del SDK público.

## Repos relacionados
- SDK público: `terpa-usb-host-sdk`.
- Flujo de cobro asociado: `terpa-pay`.

## Inicio rapido
```bash
git clone git@github.com:terpa-solutions/terpa-developer-docs.git
cd terpa-developer-docs
npm install
npm run start
```

## Estado
Activo y público.
